# Bounded semantic memory, not transcripts

## 问题

- 跨 Session 的记忆完整应保存完整聊天，还是保存可行动语义？

## 选择

- 默认保存有界但语义完整的目标、Decision、Fact、Evidence、进展、下一步、阻塞与风险；原始聊天仅在用户显式要求时作为私有 Artifact。

## 拒绝

- 默认保存完整聊天或思维链

## 为什么不选

- 完整 transcript 会增加隐私、存储和上下文腐烂风险，不是可靠恢复所需的最小充分状态。
