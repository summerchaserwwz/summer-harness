# Provisional GUI delivery shape

## 问题

- GUI 使用本地 Web、原生桌面还是共享前端分阶段交付？

## 选择

- React 本地 Web 首发，Wails 薄壳后续交付

## 拒绝

- 仅浏览器且永不提供桌面壳
- Tauri 或 Electron 首发

## 为什么不选

- 共享 Web UI 能最快提供完整看板；Wails 可复用 Go Core；首发桌面打包会提前引入签名、公证和多平台成本。
